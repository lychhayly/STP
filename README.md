# Instalación-Termux:

* `apt update && apt upgrade -y`
* `apt install git -y`
* `git clone https://github.com/DeepSociety/STP`
* `cd STP`
* `bash install-termux.sh`
* `bash stp.sh`

# Instalación-Debian:

* `sudo apt-get update && sudo apt-get upgrade -y`
* `sudo apt-get install git -y`
* `git clone https://github.com/DeepSociety/STP`
* `cd STP`
* `bash install.sh`
* `bash stp.sh`
